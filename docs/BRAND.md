# Brand assets — placeholders + how to generate the finals

Phase 1 ships **functional placeholders** so the app is installable and links unfurl today. They're
generated by a pure-Node, zero-dependency rasterizer (`scripts/gen_icons.mjs`) — a blue lightning bolt
(`#58a6ff`) on the app's dark background (`#0d1117`) with a subtle depth disc.

Regenerate the placeholders any time:

```
node scripts/gen_icons.mjs
```

## Asset set (paths are load-bearing — keep them stable)

| Path | Size | Used by |
|---|---|---|
| `icons/icon-192.png` | 192×192 | PWA manifest (`any`) |
| `icons/icon-512.png` | 512×512 | PWA manifest (`any`), splash |
| `icons/icon-maskable-512.png` | 512×512 | PWA manifest (`maskable`) — bolt kept inside the ~80% safe zone |
| `icons/apple-touch-icon.png` | 180×180 | iOS home-screen (opaque; iOS applies its own mask) |
| `icons/favicon-32.png` | 32×32 | browser tab favicon |
| `og/og-default.png` | 1200×630 | Open Graph / Twitter card (site-wide) |

**To swap in finals:** produce PNGs at these exact paths + sizes and commit them. No code changes
needed — `index.html` and `manifest.json` already reference them. (The maskable icon must keep its
important content within the center 80% "safe zone" or Android will crop it.)

## Generation prompts (for an image model — e.g. Midjourney / DALL·E / Ideogram)

Brand: **Outage Atlas** — a calm, trustworthy, privacy-first national power-outage map. Palette:
background `#0d1117` (near-black blue), accent `#58a6ff` (sky blue), positive `#3fb950` (green),
text `#e6edf3`. Tone: clean, modern, civic-utility — *not* alarmist. Flat/geometric, not skeuomorphic.

### App icon (square, for `icon-192/512` and `apple-touch-icon`)
> A minimal flat app icon on a dark near-black background (#0d1117). A single bold sky-blue (#58a6ff)
> lightning bolt, centered, geometric, crisp edges, subtle soft radial glow behind it. Slight rounded-
> square framing. No text, no gradients beyond a faint center glow. Modern civic-tech utility app icon,
> high contrast, vector-clean. 1024×1024.

### Maskable icon (Android adaptive — keep art in the center 80%)
> Same sky-blue lightning bolt on a full-bleed dark #0d1117 field, but the bolt sized smaller so it sits
> entirely within the central 80% safe zone (Android crops the edges into a circle/squircle). Even
> margins on all sides. No text. 1024×1024.

### OG / social card (`og/og-default.png`, 1200×630)
> A 1200×630 social share banner, dark #0d1117 background. Left third: a large sky-blue (#58a6ff)
> lightning bolt inside a soft dark disc. Right two-thirds: the wordmark "Outage Atlas" in a clean
> bold sans-serif (#e6edf3), and beneath it the tagline "Power outages near you — free national map"
> in muted gray (#9da7b3). A thin green (#3fb950) accent bar along the very bottom edge. Balanced,
> lots of negative space, legible as a small thumbnail. Flat, modern, civic-tech.

Per-area dynamic OG cards (live customers-out + locator per city/county) are a **Phase 5** task; this
default card is the site-wide fallback until then.
